# Chidori
千鳥 - Notification application

# Getting Started
1. Clone repo
```bash
git clone https://github.com/wizo06/Chidori.git
```
2. Install Node.js dependencies
```bash
npm i
```
3. Make a copy of the template config file
```bash
./prepare.sh
```
4. Edit the config file
```bash
nano conf/user_config.toml
```
5. Start Chidori
```bash
npm start
```
